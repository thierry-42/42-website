# 42 Codex Website Starter

This package contains the strategic, content, and implementation instructions for building the 42 HubSpot consultancy website.

## Files

- `AGENTS.md` — persistent repository instructions for Codex
- `docs/42_WEBSITE_MASTER_BRIEF.md` — full positioning, sitemap, page content, design, and technical brief
- `docs/WORKFLOW_FIGMA_CODEX_GITHUB_DEPLOYMENT.md` — practical setup and delivery workflow
- `src/content/site-content.json` — structured starter content and placeholder governance

## First Codex prompt

Read `AGENTS.md`, `docs/42_WEBSITE_MASTER_BRIEF.md`, `docs/WORKFLOW_FIGMA_CODEX_GITHUB_DEPLOYMENT.md`, and `src/content/site-content.json`.

Then initialise the website repository according to the brief. Start with the project foundation, scripts, design tokens, content types, shared layout primitives, header, footer, and placeholder asset structure. Do not invent proof, client names, partner status, booking links, or metrics. Run lint, type checking, tests, and a production build before finishing.
